Sticky Windows

Sticky is a small game I made. The goal of the game is to move a cube through multiple mazes without touching the walls and avoiding other obstacles. Moving the cube is done by using the W, A, S, and D keys. Currently, the game is only available for Windows PCs, but you can run it on MacOS and Linux using WINE.

For more information, visit the Sticky Github repository.

PSXControllerTest PS1

PSXControllerTest is a program I made for the original Sony Playstation. This program will tell the user which button they are currently pressing. Only controllers without analog sticks are supported, and the program only reads inputs from controller port 1.

I wrote this program to help me familiarize myself with programming for the PSX. As a result, it is very bare-bones.

This was written with the help of a tutorial series by Wituz. As a result, some of the code is his (namely constants.h and controller.h). The tutorials can be found here and the files for the tutorials are here. If you want to start programming for the PSX, I recommend taking a look at the PSXDEV website.

For more information, visit the PSXControllerTest Github repository.
